Missing some calendars?

If there are too few calendars visible above, try the following steps:

  1. Maximise your Chrome window as the extension may not find all calendars if there is not enough space for the calendars list.
  2. Go to Google Calendar and let the page load fully before you go to another tab.
  3. Ensure that the list(s) of your calendars are visible (unfolded). If they weren't, unfold them and refresh the Google Calendar page.
  4. Finally, refresh this options page.

If this doesn't help, try increasing the configuration value "Delay when hiding the drawer (in ms)" below to a larger number, e. g. to 1000.

You can also lower this value in order to speed up the switching process in case you're not experiencing issues.

Should your problem persist, feel free to reach out to the developer at multiplecalendarsselector@tarasiuk.me.
